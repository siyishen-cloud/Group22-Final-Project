# NYC Flights Analysis App

## How to run this app locally

1. Ensure you have R and RStudio installed.
2. Download the 'app.R' from this repository.
3. Open 'app.R' in RStudio.
4. Install the required packages by running the following command in the R console:
   install.packages(c("shiny", "ggplot2", "dplyr", "nycflights13"))
5. Click the "Run App" button at the top right of the script editor in RStudio.
